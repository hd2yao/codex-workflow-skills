## 错误学习

| 字段 | 内容 |
|---|---|
| 触发信号 | 日报只报告 YA FundMind daily/weekly 成功，没有说明 V2 Final 目标正在等待每日运行、由谁监控、何时自动恢复以及等待期间继续什么。 |
| 原始目标 | 所有长期目标在等待外部条件时仍有明确监控、自动恢复或人工提醒，并尽可能切换到安全并行工作。 |
| 错误假设 | 周期任务健康等于目标连续性；目标线程里的 heartbeat 信息不需要进入任务账本；一个 gate 等待可以代表整个目标 blocked。 |
| 错误类型 | 反复返工、数据源割裂、流程过轻。 |
| 影响范围 | 跨任务目标、日报、自动化、用户对当前进度和后续安排的可见性。 |
| 防复发规则 | 外部等待必须登记 follow-up；有可并行轨道时不全局阻塞；自动续作必须校验 Automation 和原线程；日报必须展示等待、监控、恢复和用户操作。 |
